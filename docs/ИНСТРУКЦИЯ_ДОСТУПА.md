# Инструкция Доступа

## Основные адреса

- сайт: `https://n8ndo.es`
- `n8n`: `https://n8n.n8ndo.es`
- `scanner`: `https://scanner.n8ndo.es`
- `OpenClaw`: `https://openclaw.n8ndo.es`

## Как заходить в n8n

1. Открой `https://n8n.n8ndo.es`
2. Введи логин `daniel`
3. Пароль хранится machine-local и не записывается в репозиторий:
   - `C:\Users\wiwal\.ai-global\n8ndo-hetzner-secrets.env`

## Как заходить в OpenClaw

1. Открой `https://openclaw.n8ndo.es`
2. Откроется `OpenClaw Control`
3. Если интерфейс попросит gateway token, бери его только из machine-local секрета:
   - `C:\Users\wiwal\.ai-global\n8ndo-hetzner-secrets.env`

## Быстрая проверка, что всё живо

- `n8n` открывается по HTTPS
- `OpenClaw health` отвечает по:
  - `https://openclaw.n8ndo.es/healthz`
- `scanner` открывается по своему домену

## Важно

- секреты не копировать в репозиторий
- пароли и токены держать только machine-local
- production-сервер сейчас находится на `Hetzner`, не на `DigitalOcean`
